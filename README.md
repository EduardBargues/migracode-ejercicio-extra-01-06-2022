# migracode-ejercicio-extra-01-06-2022

Dada la siguiente base de datos, haced una api que soporte las siguientes operaciones:

- GET /users
- GET /users/{userId}

- POST /users
- PUT /users/{userId}
- DELETE /users/{userId}
